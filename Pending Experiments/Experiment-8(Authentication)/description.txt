#  Role-Based Access Control (RBAC) System using Node.js, Express & JWT

This project demonstrates how to build a **Role-Based Access Control (RBAC)** system using **Node.js**, **Express.js**, and **JWT (JSON Web Token)** for secure authentication and authorization.

It allows you to create and log in users with different roles (Admin, Moderator, and User), and restrict access to routes based on their assigned role.

---

##  **Objective**

To understand and implement:
- User authentication with JWT tokens.
- Role-based authorization middleware.
- Secured APIs that only specific roles can access.
- Error handling for invalid tokens and insufficient permissions.

---

##  **Technologies Used**

- Node.js  
- Express.js  
- bcryptjs (for password hashing)  
- jsonwebtoken (for token generation)  
- dotenv (for environment variables)  
- nodemon (for auto-restart during development)
